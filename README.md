# Airport Baggage Handling System (ANDB)

## Overview
This project focuses on **creating and managing system requirements using Jira** for an **Airport Baggage Handling System** as part of the **ANDB (Advanced Database Systems)** subject.

The work is organized into **time-boxed sprints**, where Jira tasks are used to design, validate, and implement a database-driven baggage management solution using **SQL Server** and **Power BI**.

---


## Objectives
- Design and implement Airport database
- Conect sql server to Power BI
- Create summaries and reciept template and generation

---

## System Scope: Baggage Handling at an Airport
The baggage handling system covers the lifecycle of baggage from **check-in to aircraft loading and arrival reclaim**.

### Key Processes
- Passenger baggage check-in
- Baggage tagging and tracking
- Aircraft loading and unloading

---

## Jira Usage
This project uses Jira to plan and track sprint-based development activities.

### Sprint Information
**Sprint Name:** Airport Sprint 1

**Sprint Duration:** 4 December – 18 December

**Sprint Goal:**
Within this 2 week sprint deliver the first functional foundation of the baggage management system by enabling the team to register validate store and retrieve baggage data and also to provide initial visibility of baggage data through basic dashboards

### Sprint Tasks
- Design baggage database schema
- Database registration of checked-in baggage
- Baggage weight validation
- Baggage loading list generation
- Develop baggage location dashboard UI
- Passenger receipt and confirmation generation
- Build flight statistics reporting module

---

## Tools and Technologies

- Jira – Used for sprint planning, task tracking, and requirement management
- Microsoft SQL Server – Used for designing and implementing the baggage handling database
- Power BI – Used for creating dashboards and reports for baggage location and flight statistics
